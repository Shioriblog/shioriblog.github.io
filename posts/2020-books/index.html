2020年读书总结

2020-12-22
Yearly

2020年虽然因为疫情在家，但整体就没怎么读书，而且读书效率非常不均衡，有好几个月不读一本书的时候，也有一个月读快20本书的时候。读书依然非常杂。往年年末会分虚构和非虚构来总结，今年就总结在一起，记录一下今年读过的，比较喜欢的书（有中文版的就会放中文版）。

另外10月份开始，和两位友邻制作了一个只是安利书，还是以外文虚构为主的播客。年末在播客上我们也总结了一些今年喜欢的和不喜欢的书，播客里提到过的这里仅仅简略提一下。

非虚构

谁在看中国画

[英]柯律格（Craig Clunas） / 2020 / 广西师范大学出版社

这本书一月份读了英文，后来发现中文版也出版了。这本书不是一本面向普通读者的科普类中国艺术史著作，而是一本有问题意识的学术著作。在作为西方传统学科的艺术史之中，“中国绘画”中的“中国”是什么？“中国”这个概念是如何形成的？柯律格运用大量史料和物证，分析大量绘画来证明他的论点。是一次十分有趣的阅读体验。

鲁迅杂文集

鲁迅 / 2015 / 天津人民出版社

重读鲁迅，这本集子选的很好，其中能读到冷嘲的鲁迅，愤怒的鲁迅，毒舌的鲁迅，讽刺的鲁迅，有幽默的鲁迅，有生活化的鲁迅，有讲台上的鲁迅。并且即使到了21世纪，鲁迅当年批评的问题，依然存在，像鲁迅这样犀利的作家，依然被需要。

Recoding Gender

Janet Abbate / 2012 / The MIT Press

不是说这本书写的有多好看，但很开眼界。作者研究了二战前后，在美国和英国，计算机发展过程中，（白人中产）女性的关键作用。（白人中产）女性是如何一开始被认为适合从事计算机编程工作，到后来乃至现在对编程工作的刻板印象就是码农（男）。读这本书可以了解这段历史，很有意思。

Animal Behaviour:

Tristram D. Wyatt / 2017 / Oxford University Press

不了解动物行为学，这本牛津通识读完算是一个简单的科普，很有意思，读起来像小时候看百科全书，又像看动物世界。对动物行为学感兴趣的话，推荐读这本作为科普。

This Brilliant Darkness

Jeff Sharlet / 2020 / W. W. Norton Company

这是今年读到的很喜欢的一本非虚构，作者因为心脏病加不眠症，开始了晚上去采访夜晚工作的人，用手机给他们拍照，用文字写下他们的故事，那是一些真实又另人心痛的故事，是处于“社会黑暗面”的陌生人，是所谓的“底层人们”，是我们不愿意去多加了解去倾听的人们的故事，是那些街上的流浪汉，是和我们擦肩而过的人，是犯罪分子，是毒瘾患者，是极右，是新/法/西/斯。作者讲述的故事，非常值得一读。

<おんな>の思想 私たちは、あなたを忘れない

上野 千鶴子 (著) / 2013 / 集英社

这是一本上野千鹤子的文章集，分为两个部分，第一部分介绍70，80年代日本的女权，第二部分是西方的女权和后殖民经典书籍的导读（比如福柯的性史，东方主义，can the subaltern speak等）。我非常喜欢第一部分。第一部分写了好几位日本女作家女权主义者先驱性人物，都是我不知道的，并且想要更多了解的。可惜介绍的那些人的书都很难读到。。

东京百景

[日] 又吉直树 / 2020 / 上海译文出版社

这是一本又吉直树的散文集，题目说是东京百景，其实只是致敬了他喜爱的太宰治，文章并非写城市或者写景，而是通过东京的地名写人写故事。众所周知又吉直树本职是搞笑艺人，我喜欢读他写搞笑艺人的那些文章，可以看到后来《火花》的故事的原型。有一些故事他上ametalk之类的谈话节目也说过好几遍。这本书我读的是日文版，中文版我也翻了一下，不推荐中文翻译。

言い訳

塙　宣之 / 2019 / 集英社

推荐给所有喜欢漫才的人和M1饭。除了塙さん的分析了当红的漫才组合（唯一批评了三四郎ww），能让人看到漫才师们的热情专注和努力，真是一生悬命来搞笑，认真态度也非常令人感动，并且热血沸腾！

Feminist City

Leslie Kern / 2020 / Verso

这本书在博客上聊过就不多说了。是一本女权主义加城市研究，又有很多作者个人经历的书，读起来非常有趣，十分推荐。（播客节目列在最后）

素人之亂

松本哉 / 2012 / 推守文化

前几年读了《世界大笨蛋反叛手册》知道了松本哉，后来在youtube上看了纪录片《素人之乱》，今年读了这本早期的书。松本哉实在是太逗了！游/行/示/威都能这么无厘头，让人看着就想加入他的运动中去！

无政府主义人类学碎片

[美] 大卫·格雷伯 / 2014 / 广西师范大学出版社

因为今年大卫·格雷伯去世，所以读了这本书，这本应该和松本哉的《素人之乱》和《世界大笨蛋》一起读。我自己读起来非常感同身受！

资本社会的17个矛盾

[美] 大卫·哈维 (David Harvey) / 2016 / 中信出版社

前一大半（尤其第一部分）算是补了政治经济学的课，可以当作简单的入门。后半部分读起来就比较熟悉。最后提出的几条政治构想读起来也是感慨：要全民都能住得起房，上得起学，就得起医；最大化利用人工智能减少分工解放人类劳动力；保护环境；减少人的异化等等，多么美好的未来图景。我们能做到吗？我们还有时间去斗争吗！！！

父权制与资本主义

[日] 上野千鹤子 / 2020 / 浙江大学出版社

一本关于马克思女权的科普书。虽然学术书的写法读起来不是特别友好，但说的道理都简单易懂。日文书是30年前左右出版的了，30年后被介绍到中国，反而可以作为面向大众的桥梁类书籍。

新左運動與公民社會

安藤丈將 / 2018 / 左岸

围绕“日常性”重新legitimatize日本新左翼运动，试图从盖棺定论的新左翼=暴力，恐怖，极左这个论点上，写出新左翼的另外一个历史，并探讨这个“失败”的历史的遗产是什么，对311后日本社会的运动的贡献是什么。这是作者的博士论文改写的。问题是“日常性”这个概念太单薄，不足以还原复杂的左翼运动的历史。但是最后联系当下，对未来社运的还是很有启发性。最好看的是前面两篇序，和最后一篇译者以台湾人反思台湾社运的立场写的文章。

日本語全史

沖森卓也 / 2017 / 筑摩書房

一本关于“日语的历史”的语言学+历史的书，非常有趣，写的也是简洁明了，有点“大家小书”系列的味道。知道很多不知道的日语词语用法背后的故事，有趣。

Great Maps

Jerry Brotton / 2014 / DK

好看的画册，知道了很多不同的地图，虽然看完也不能说都记住，但喜欢作者不是用一种发展进化的观点讲述地图的历史，不同时代不同文明通过真实的地图或想象的地图反映了当时人们对自己身处的世界和空间的认识。

The End of Everything

Katie Mack / 2020 / Simon & Schuster

今年最喜欢的一本书。在播客上提到好几次了就不多说了。这是一本关于宇宙将会怎么灭亡的科普书，语言生动幽默，讲得深入浅出很容易理解。也没有什么人生鸡汤，科普的知识也很前沿，非常好看。

跟风读的：

走出唯一真理观

陈嘉映 / 2020 / 上海文艺出版社

这本书很红了。不多说了。我是第一次读陈嘉映，觉得他写作文章还是很真诚的一个人，而且他还在继续发声就是可贵的（我对中老年体制内搞学术男性有很大的偏见）。但这本书应该适合更年轻时候读，我现在读起来感觉新鲜东西不多了。

把自己作为方法

项飙 吴琦 / 2020 / 上海文艺出版社

这本也是今年特别红的，也是我第一次读项飙，我个人不是很喜欢，感觉他说的很多我都不能认同。

虚构

小说

雨

[马来西亚] 黄锦树 / 2018 / 四川人民出版社

在播客上聊过了，最喜欢的书之一。非常喜欢黄锦树，尤其喜欢他的语言。明年想继续读他的书。

The Bear

Andrew Krivak / 2020 / Bellevue Literary Press

今年最喜欢的小说之一。一本末世小说，语言特别美，写的也很童话，对人类的灭亡也没有焦虑感，特别喜欢。播客上有聊到过。

Summer

Ali Smith / 2020 / Hamish Hamilton

今年最喜欢的小说之一。期待好久，终于！终于！Ali Smith的四季四部曲最后一本出版了！这本书在播客里聊了好几次就不多提。Ali Smith真是语言好棒，又聪明。非常非常推荐！

秧歌

张爱玲 / 1968 / 皇冠出版社

今年最喜欢的小说之一。在播客上聊过了。尤其喜欢张爱玲写传统男性的革命叙事中被忽视的女性的遭遇。

推し、燃ゆ

宇佐見 りん / 2020 / 河出書房新社

今年最喜欢的小说之一。在跳岛fm上聊过了。一本关于追星的小说。除了读起来非常感同身受之外，也感叹于20岁作者的才华。期待她的新作。

Sisters

Daisy Johnson / 2020 / Riverhead Books

今年最喜欢的小说之一。在博客上聊过，但应该会专门聊一期Daisy Johnson。喜欢她语言，喜欢她写母女关系。这本小说是哥特+悬疑，继承并拓展了哥特文学的传统，悬疑也很有意思，非常推荐。

A kind of spark

Elle McNicoll / 2020 / Knights Of Media

今年最喜欢的小说之一。在播客上有专门一期聊。一本关于自闭症小女孩，关于女巫，关于校园欺凌，关于不同的middle grade的小说。非常非常推荐！

火中遗物

[阿根廷] 玛丽安娜·恩里克斯 / 2020 / 外语教学与研究出版社

读的英文版，非常喜欢，已在播客上聊过。一本恐怖小说集，写阿根廷当下社会和政治，写作为一个女性在这种社会中的恐怖遭遇。一种哥特小说遇上拉美文学的感觉，非常棒，期待作者明年的新书！

素食主义者

(韩)韩江 / 2013 / 重庆出版社

这本书也在播客里聊到了。读完之后留下非常深刻的印象。写对女性，对女性身体的压迫，切入点非常好，写的方式也非常独特。英文版翻译读起来很好（虽然有人说和韩文出入很大？），推荐。

How to be both

Ali Smith / 2014 / Hamish Hamilton

今年读了一本我喜欢的Ali Smith早期的书，这一本非常有她的风格，意识流，碎片地描写，简介诗意的语言，充满witty的文字游戏和reference，是一本令人愉悦，又让人思考，又充满野心的作品。明年继续读Ali Smith以前的书。

外面是夏天

[韩]金爱烂 / 2019 / 人民文学出版社

读的是日文版，因为特别喜欢《你的夏天还好吗》所以买了书来读，但是，主题和写法都大同小异读起来非常审美疲劳。和作者自己生活贴近的写得比较真实，其他感觉技巧性太强，过于匠气。读过《你的夏天还好吗》再来读这个，失望略大。日文版翻译很好。但还是期待她的新作。

诗 我不怎么会读诗，也读的不多，今年读了两本，都很喜欢。

Conflict Resolution for Holy Beings

Joy Harjo / 2017 / W. W. Norton & Company

语言很好，像原住民那样讲故事的感觉，但又很融洽并且聪明地把原住民的文化和现代北美文化结合在一起。第一次读Joy Harjo的诗，喜欢。

Dearly

Margaret Atwood / 2020 / Vintage Publishing

玛格丽特·艾德伍德的新诗集，语言很平常但优美，很多我也读得懂，有一些甚至特别喜欢，尤其Dearly那一首，将不再经常被使用的Dearly一词(It’s an old word, fading now")和逝去的事物，年华，日子(如珍珠一般的“How to keep track of the days?/ Each one shining, each one alone, /each one then gone”)联系在一起，美丽忧伤。

名著 今年的目标是多读一些名著，但结果还是读的不多，明年继续：

一个青年艺术家的画像

[爱尔兰] 詹姆斯·乔伊斯 / 2009 / 浙江文艺出版社

读的英文版，喜欢乔伊斯的语言，除了意识流的写作方式外，书中各种隐喻也非常有野心，少年的成长，对宗教和传统的叛离，对艺术的追求，男主是乔伊斯的对照，而且姓Daedalus这个希腊神话中的著名工匠，是否可见乔伊斯的艺术观呢？

双城记

[英] 查尔斯·狄更斯 / 2015 / 作家出版社

读加听的英文版，我不是很喜欢这一本，喜欢书中写群像戏，但对法国大革命的描写不敢苟同。

包法利夫人

福楼拜 / 2003 / 人民文学出版社

特别喜欢包法利夫人这个故事！我大概就是喜欢出轨的故事😂 而且让我读着觉得萨冈的《狂乱》绝对有致敬包法利夫人！另外这本书我是先读英文版，然后书中有掉页，就去日文翻译，日文翻译的与其太好笑，又去读了中译本。感觉，英译本，日译本和中译本是三本不同风格的书😂

霍乱时期的爱情

[哥伦比亚] 加西亚·马尔克斯 / 2012 / 南海出版公司

啊！好厉害的小说。是关于人的小说，时间，死亡，爱情，各种爱情，各类人，不同阶级不同肤色。一个男人年轻时候对一个女孩表达爱意，50多年之后，当两人都经历了人生和爱情的风风雨雨之后，他再次对她表达爱意。最后老人之间的爱情写得真好，对老人的身体性欲写的特别好，另外警句好多啊。但我不是很喜欢这个故事。

螺丝在拧紧

[美] 亨利·詹姆斯 / 2020 / 上海文艺出版社

读的英文版，有意思的哥特小说，在播客里聊过了。亨利·詹姆斯的语言风格真是奇特。。
